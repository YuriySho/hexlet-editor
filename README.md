## Installation

```bash
$ make install
```

## Running the app

```bash
$ cd backend

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
$ cd backend

# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

localhost:3001

localhost:8081 frontend


## Frontend

```bash
$ cd frontend

# install
$ make install

# develop
$ make develop

# build
$ make build
```

## Known issues

При работе с фронтендом в редакторе VS Code открываем не всю папку с проектом, а только frontend, поскольку расширение редактора eslint будет искать свои плагины в корне проекта.
